Common Page Layout

The common Layout for my page was the fixed menu header. This header can be seen across all pages. This header is useful navigating through all the different features of my project.


The different Navigation links are as follows:

  • Movie Trailers Drop Down - One can navigate to different links such as Upcoming, Top Rated, Now Playing and Popular Movie Trailers.

  • TV Trailers Drop Down - This navigation provides links to Top Rated, On Air and popular TV Show Trailers.

  • Search Trailers Drop Down - This drown allows one to search for movies, tv shows and theatre addresses.

  • News Feed Drop Down - Two News Feeds about TV and Movie are provided in this dropdown.
