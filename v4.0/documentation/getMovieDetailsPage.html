Get Movie Details Page

This page contains all the details about the movie. It contains information such as plot of the movie, cast, crew, reviews and Facts such as Budget, release date, runtime , genre and revenue. This page also provide me with the rating of the movie that movie db provides. I also provide an imdb link of the movie for the user to view the imdb page of the movie.



This page can be accessed by clicking the Get Details Button for the movie that the user wants to review. The different parameters for this API are credits for getting cast and crew and id of the movie for getting overview, ratings and series Facts. Reviews are Fetched using the reviews parameter and id of the movie.
