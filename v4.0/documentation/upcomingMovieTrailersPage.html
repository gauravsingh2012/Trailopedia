Upcoming Movie Trailers Page

The page displays all the upcoming movie Trailers which are about to release. The movie db api url to obtain the result set is https://api.themoviedb.org/3/movie/upcoming?api_key=052ca14131fc2be845560a604c66ffd5. The result set is parsed and one div block is cloned with respect to the number of elements in the response from the api.


Note that in the above screen shot I have Get Details and Watch All Trailers Button as well. The Get Details brings details about the movie and watch all trailers button allows access to all trailers for the movie.I have also provided pagination at the end of each page to parse all other json pages response by providing a page parameter in the API. Also Observe that the page is responsive
