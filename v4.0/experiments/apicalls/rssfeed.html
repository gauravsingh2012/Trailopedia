Purpose

The purpose of this experiment is to call the New York times RSS Feed using the Google Feed API

Experiment

The Experiment calls the New york times api to retrieve the rss feeds. This experiment is used to understand how to generate rss using the Google Feed API. These feeds can be generated in a very time as google allows developeres to use the automated code to generate feeds.

We have used javascript completely to generate dynamic elements of the feeds. We display the url feeds by calling the feed method of the google jsapi. This automatically fetches the data and we dynamically generate the elements as a href. I have also provided feed limit as well which helps to work with less data and avoid congestion in the traffic.
