Purpose

The purpose of this experiment is to understand responsive design in nested divs. Here I have a main container div and inside this container div, I have two div nested divs. One div takes 60% width of the main container div and the other takes 30% of the main container width. The rest 10% I use for providing margins for the look and feel. To make these divs responsive, I found out that one can use the display : table css style to make it responsive

Experiment

This experiment can be seen as starting base for customizing my personal webpage. One can resize the window and see that the second nested div actually starts going down as you resize the window.
