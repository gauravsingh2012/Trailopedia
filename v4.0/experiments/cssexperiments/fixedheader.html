Purpose

The purpose of this experiment is to make a fixed navigation header with the help of CSS. This experiment also requires to show a drop down menu.

Experiment

Notice the fixed black header when you scroll down. The header remains moves down with the scroll Also notice when you hover over the Drop Down menu , you will see a drop down menu slide down. I used this Note that I used jQuery to achieve this, but I can also use CSS hover property to display these menus. I wanted a slide down animation with jQuery, so I used jQuery slide Down and Slide Up Property. Note that the fixed header looks like bootstraps fixed header but I am only using the design idea. The CSS is mine :).
