Purpose

The purpose of this experiment is to use the google Maps API. Using this API one can embed the google maps in their website and use it to see diffrent locations.

Experiment

Observe that I have provided a fixed value to provide input to the Google Maps. The use case is that I will have a fixed value in a contact us page and will use that value to show my location. I can also generate this by taking input from the user. But since, it is not one of my use cases to generate locations by providing user input, hence I decided to work with a fixed label, extract its values and provide input to the Google Map API.
