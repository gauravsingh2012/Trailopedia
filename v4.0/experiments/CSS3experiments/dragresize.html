Purpose

The purpose of this experiment is to use jquery UI for Draggable divs and resizable divs.

Experiment

Drag the red div block and play around with it. Resize the Blue div block.

We make the divs draggable by calling the .draggable() on a div block and for resizing we use .resizable(). The jquery code used for making divs draggable and resizable is as follows: 

                            $(document).ready(function () {
                                $(".exp-drag-divs").draggable();
                            }); 
                            $(document).ready(function () {
                                $(".exp-resize-divs").resizable();
                            });
