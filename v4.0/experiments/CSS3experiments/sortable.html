Purpose

The purpose of this experiment is to use jquery UI for making sortable lists.

Experiment

Drag the Lists and try and reposition them amongst themselves. This can be achieved by defining the following HTML: 

                            <ul class="sort-lists">
                                <li>Sortable List 1</li>
                                <li>Sortable List 2</li>
                                <li>Sortable List 3</li>
                            </ul>

We make the lists sortable by calling the .sortable() on the ul class. The jquery code used for making lists sortable is as follows: 

                            $(document).ready(function () {
                                $(".exp-sortable-lists .sort-lists").sortable();
                            });
