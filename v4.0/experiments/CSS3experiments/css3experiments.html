Purpose

The purpose of this experiment is to work with CSS3 properties like @font-face and radial Gradients

Experiment

Observe the radial gradient background with spirals in the center. The radial Gradient produces the spirals based on the center positions. There is a font change in the experiment. The font is a downloaded Pacifico font which can be used by setting the @font-face property in the CSS.

Hove Over the buttons to see web-transition effects. Another really coll CSS3 property that I observed that allows us to have a delayed effect on hover.
